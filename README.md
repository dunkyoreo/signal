# 🧠[SIGNAL](https://dunkyoreo.github.io/signal/)📡

A single-page attention diagnostic disguised as a control-room instrument panel.

## What it does

Four short readings, each catching a different way focus drops out:

- **Signal Lock** — simple reaction time
- **Signal Watch** — sustained attention (catch the X in a letter stream, decay tracked over time)
- **Crosstalk** — selective attention under interference (Stroop-style color/word conflict)
- **Frequency Hop** — cognitive flexibility (rules flip mid-stream, cost is measured)

Ends in a composite score, a plain-language breakdown of what each reading actually caught, and a full glitch/static/boot-sequence treatment on top.

## Stack

Single HTML file. No dependencies, no build step, no backend. Vanilla JS, canvas for the scope and static effects, CSS for everything else.

## Why this exists

Built as an informal test of how faithfully an AI can execute an open-ended creative idea across iteration — from first concept, through UX and aesthetic feedback, to a full animation pass — without the concept drifting or the codebase falling apart along the way.
